# Desafio Senior Sistemas William Roberto

Documentação do Projeto

## Descrição

O projeto foi construido utilizando Angular para o front end e Java Spring para o back end sendo o Mysql o banco de dados da aplicação.

## Tecnologias Utilizadas

- Java 17 ou superior;
- Spring Boot;
- MySQL;
- node.js;
- angular;
- Linguagem typeScript;

## Funcionalidades

- Pagina inicial;
- Painel dos Clientes (Cadastrar, Listar, Atualizar, Deletar);
- Painel dos Contatos (Cadastrar, Listar, Atualizar, Deletar);

## Estrutura do Projeto

O projeto foi estruturado no mesmo repositorio, sendo um diretorio para o back end: `cd crm-senior-backend-project` e o outro para o front end: `cd crm-senior-frontend-project`

## Instalação e Uso

### Requisitos

### Back End 

- Java 17 ou superior; 
- mavem (configurado nas variaveis de ambiente);
- MySQL 8 (podendo ser um XAMPP por exemplo);

### Front End

- node.js;
- angular;

### Passos para Instalação

### Baixando o projeto

1. Clone o repositório: `git clone https://github.com/williams181/Desafio-FullStack-Senior-Sistemas-William-Roberto`;
2. Entre na pasta do projeto: `cd Desafio-FullStack-Senior-Sistemas-William-Roberto`;

### Execução do projeto

### Back end

1. Acesse o diretorio do projeto e com a IDE de sua preferencia e abra o projeto `cd crm-senior-backend-project`;
2. Configure o arquivo `application.properties` presente no projeto Java, com as informações do seu banco de dados;
3. Execute o back end da aplicação inseridno o seguinte comando no terminal: `mvn spring-boot:run`;

### Front end

1. Acesse o diretorio do projeto e com a IDE de sua preferencia e abra o projeto `cd crm-senior-frontend-project`;
2. Execute o front end da aplicação inseridno o seguinte comando no terminal: `npm install`;
3. Execute o front end da aplicação inseridno o seguinte comando no terminal: `npm install -g @angular/cli`;
4. Execute o front end da aplicação inseridno o seguinte comando no terminal: `ng serve --open`;
